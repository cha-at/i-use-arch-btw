# "I use Arch btw" language

**"I use "I use Arch btw" language btw."**

"I use Arch btw" language 是一款图灵完备的、极简主义的、基于梗的编程语言（这是废话）。基于Brainfuck ，但剔除了那些无聊的符号，转而使用更符合 ***“高级黑客”*** 身份的短语。
使其成为**世界上最高级的语言。**

> [!IMPORTANT]
> 太高级了我操。

## 此处输入二级标题

语法非常好懂:

| 指令 | 对应 BF | 逻辑描述 |
| --- | --- | --- |
| `i` | `<` | 指针左移 |
| `use` | `>` | 指针右移 |
| `btw` | `+` | 当前单元格 +1 |
| `the` | `-` | 当前单元格 -1 |
| `arch` | `.` | 输出 ASCII 字符 |
| `linux` | `,` | 输入字符 |
| `by` | `[` | 循环开始 |
| `way` | `]` | 循环结束 |
